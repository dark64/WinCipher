# WinCipher

A lightweight utility that adds secure file encryption directly to your Windows context menu.

## Features

- **Right-click encryption**: Simply right-click any file and select "Encrypt/Decrypt" to protect your sensitive files
- **AES encryption**: Uses industry-standard AES encryption to secure your data
- **Secure deletion**: Includes SDelete integration to securely wipe original files after encryption, preventing recovery

## Installation

1. Download the portable package
2. Extract the contents
3. Run `install.bat` as administrator
4. The "Encrypt/Decrypt" option will now appear in your right-click context menu

## Usage

1. Right-click any file you want to encrypt or decrypt
2. Select "Encrypt/Decrypt" from the context menu
3. Enter your password when prompted
4. The file will be encrypted or decrypted accordingly
